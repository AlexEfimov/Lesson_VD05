О сайте

Этот сайт - иллюстрация того, как можно использовать Flask вместе с шаблонизатором Jinja

Он состоит из:

  • Основной шаблон с панелью навигации (base.html).
  • Домашней странице расширяющей основной шаблон (home.html).
  • Страницы "О сайте" расширяющей основной шаблон (about.html).

Flask — это микро-фреймворк для разработки веб-приложений на языке Python. Он называется "микро", потому что не принуждает разработчика использовать определённые инструменты или библиотеки. Flask предоставляет базовый набор возможностей, оставляя разработчику свободу выбора остальных компонентов.

Jinja — это один из самых популярных шаблонизаторов для языка Python. Он используется для генерации HTML, XML или других текстовых форматов из шаблонов. Jinja позволяет отделить логику приложения от его представления, что делает код более организованным и читаемым.

Для создания панели навигации был исподьзован Bootstrap 5.3
